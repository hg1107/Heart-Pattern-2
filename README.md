## How to Run

1. Install Python 3.
2. Install Jupyter Notebook:

   ```
   pip install notebook
   ```
3. Start Jupyter:

   ```
   jupyter notebook
   ```
4. Open main.ipynb.
5. Run cells sequentially from top to bottom.

---

## How It Works

* The notebook is divided into multiple cells.
* Each cell contains a specific part of the logic.
* Code execution produces immediate output below the cell.
* Variables and functions persist across cells during runtime.

This allows:

* Incremental development
* Debugging per section
* Interactive experimentation
